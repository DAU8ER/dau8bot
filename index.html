This bot is a useful chat bot for custom commands, emote reactions, timers etc. The commands are listed below. You can create custom commands in your channel (they wont be listed on this page). If you have any questions or future requests, feel free to contact me: Discord: Dauber #0001 Twitch: Dau8er If you want to JOIN or PART the bot to or from your channel, contact me or use one the respective command in the bots chat Default Prefix: 8!
Commands available for everyone:
new Commands
Description
addblacklist <your username>
Use this command If you don't want the bot to track you in this and future projects regarding the lastseen command.
removeblacklist <your username>
Use this command to undo the effects of the addblacklist command.
lastseen <username>
Command to get the last message sent message by the user in one of the bots channels
Commands
Description
afk
Sets your status to afk, you'll get greeted as soon as you write in one of the bots channels.
gn
Sets your status to sleeping, response equivalent to afk command.
tuck
Tuck someone to bed.
untuck <user> <message>
Untuck someone who just set their status to sleeping using the gn command
rafk/cafk
Resume your previously set afk status.
timer <time> <message>
Sets a timer sending your message after the specified time.
deltimer <ID>
Delete on of your timer.
getemotes
Lists all enabled emotes in the channel.
getemotec <emote>
Shows up the current Emote cooldown.
ping
Info about the Bot.
kok
Exposes your (100% real) kok length.
spam <spam count> <emote/text>
Let the bot spam your stuff.
getcmds
Lists all enabled commands in the channel.
getprefix
Shows the current channel prefix, you can always use the default prefix to use this command.
pick <your option> ; <your option>
Picks one of multiple given options. (max. 6 options) (option split key: ; )
Commands available for Moderators and the Broadcaster:
Commands
Description
addemote <emote> <cooldown>
Adds a emote to get reacted on by the Bot (10s cooldown by default).
removeemote <emote>
Removes a emote from the Bot.
emotec/cemote <emote> <cooldown>
Sets a new Emote cooldown.
addcmd <command> <message>
Adds a custom command to your channel (2s cooldown by default).
editcmd <command> <message>
Updates a custom command.
editcmdc <command> <cooldown>
Sets the cooldown of a custom command.
delcmd <command>
Deletes a custom command.
